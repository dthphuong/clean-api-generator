
# FClean API Framework
**Version 2.0**

The powerful and flexible API framework which is used Node.JS, Express and based-on **Clean architecture**
## Installation

You need to install:
- Node JS >= 12.x
- NPM or Yarn

## Run

Go to the project directory

```bash
  cd demo-clean-api
```

Install dependencies
```bash
yarn install
```

Let's rock

```bash
  node .
```
